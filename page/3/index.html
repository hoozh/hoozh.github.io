虚拟机安装 Mac OS X 记录

以后有可能涉及苹果系统上的应用开发，在硬件没配套之前，先用虚拟机尝了一下鲜：

刚开始我没多想就选择了 VirtualBox 作为虚拟机软件，参考了这篇文章《Win7 + VirtualBox安装Mac OS X雪豹操作系统图文详解》，安装时遇到黑屏的问题，搜索后发现是引导文件的问题，调整后，最后我安装 Mac OS X 需要的文件为:


  • OSX86_ModCD-031111-171757.iso (下载)

  • [苹果雪豹操作系统正式版].snowleopard10a432userdvd.dmg


按照教程安装成功后，为测试能否使用外置的UVC免驱摄像头，安装了VirtualBox 4.1.6 Oracle VM VirtualBox Extension Pack，摄像头在system profile中能识别，但是打开Photo Booth软件，操作系统立刻死机，反复尝试都是一样，包括iChat软件。打开Console工具再测试，发现只输出两行错误：
invalid pixel format
invalid context

Google无果。

只好改用安装包大小将近600MB的VMware Workstation 7，参照这篇文章《教你在VMWare下安装MAC OS X Snow Leopard 10.6》，驾轻就熟地安装完毕。

系统启动后发现一切接近完美，UVC摄像头在VMWare中可以在host和guest os之间切换，在 Mac OS X 下打开摄像头也很正常了。

问题：

  • 运行一段时间之后，找不到鼠标了
    解决：在Mac OS X 的“系统偏好设置” -> “节能器”中的“电脑进入睡眠”和“显示器进入睡眠”设置成“永不”

Silverlight for Embedded 的一些总结

这段时间在 s3c6410+WinCE6 上编写一个机械控制软件，开始使用了 Silverlight for Embedded 做界面，对于界面不复杂的软件，Silverlight 开发起来非常方便，xaml语言基本上也可以边用边学。

不过遗憾的是，我还是决定放弃了，最终还是采用SDK写界面，唯一的原因就是“太慢了”。因为我们的平台底层没有实现DDraw硬件加速，silverlight的界面渲染用的还是GDI，作为工业软件，操作起来不够快。

这里把这段时间收集的一些资源汇总一下，必备不时之需 ^^

开发工具：


  • VS2005+6410 SDK

  • Microsoft Expression Blend 2 + sp1

  • xaml2cpp 一个牛人写的小工具，绝对利器，推荐使用


教程：


  • Silverlight for Windows Embedded tutorial
    一个系列教程，作者编写了xaml2cpp

  • wince R3 silverlight编程概要


文章：


  • silverlight 模板(Template)使用

  • Silverlight 属性样式、控件模板、视觉状态

  • Styles in Silverlight

  • 布局控件快速入门StackPanel,ScrollViewer和Border

  • Silverlight examples (example source code) Organized by topic

  • WPF/Silverlight 控件的基本组成


另外值得一提的是，silverlight for windows embedded 基本上是基于Silverlight 2的，所以大部分特性可以参考Silverlight 2 的文档。
欢迎交流~~

【24W+5D】终于过糖耐了

上周四做糖筛，悲催的没过，下午电话过来医生让今天去做糖耐，不知道是不是哼哼他外婆煮的糖水还有猪蹄粥的关系，十一长假期间我的主食就是那些

于是哼爸开始小心翼翼控制我的饮食，比如水果淀粉类食物的摄取等等

星期六星期天，我甚至开始吃不喜欢吃的西红柿和黄瓜，因为据说这个可以降血糖

事实上说西红柿降血糖存在误区，西红柿里面有丰富的VC，VC据说会影响血糖测试，会呈现假阴性，如果你真相测试血糖的还是之前别吃西红柿了

黄瓜我切成细丝，加上盐和醋拌了，发现也不是很难入口，还有生胡萝卜丝，绝对锻炼刀工啊

不过在我切菜的时候，小哼哼没少在肚子里折腾，不知道是不是切菜的声音影响到他了

星期天晚上早早上床休息，第二天起床去赶公交车并走到医院挂号开单子

拿到糖水，喝下去绝对是折磨啊，总担心会吐掉，结果出去散步走走的时候，第一次吐了一小口，第二次是一大摊，不知道对吐了这么多对糖耐测试有没有影响，后来第一次抽好血下来又吐了一口，胃就没那么难受了

中间让哼爸把小熊给我们的孕妇装归还给他们，他回来后我们还上去测体重了，悲催的发现就这三四天我又重了一公斤啊，不知道运动鞋会不会有影响，可是怎么算一斤多至少的，这样是不是增重太快了啊

最后一次抽血也就是第三次已经是11：42，最难熬的就是后面等的那十几分钟，前面两个半小时多一个人去散步看小说拿哼爸手机切水果总是找到事情做，到最后等着抽血就是煎熬啊，加上肚子又饿了

抽完了，顺便问医生前两次结果，医生很好心帮我查了一下告诉我前两次血糖正常的，那么这第三次肯定不会有问题了，提起的心终于放下了

我终于吃到了早上买的面包，已经16个小时左右没有进食了，吃完出来却没觉得舒服，老泛冷汗出来感觉自己要晕倒了，和哼爸吃完午饭才觉得舒服多了，很想吃东西，吃完自己的面觉得还能吃下哼爸点的饭，不过很克制只抢了几口饭

后来去面包反买了三明治备着当下午点心，想吃生煎最后还是没去，去超市买了些蔬菜水果和哼爸一起坐公交，哼爸则中途转车回去上班开会，据说他们有技术交流会要开

回来在小区别的楼层下面捡了个陶罐，因为早上关窗户不小心把仙人掌的盆打翻弄坏了，晚上哼爸猜测可能是电砂锅的内胆，看起来很新，搞不清楚，还挺沉的。

不知道什么附体，我竟然把超市买的那些东西一件外套一个陶罐自己的背包给搬上了五楼，然后下午无聊一直在斗地主，终于再接再厉负666分到现在负三百多了

晚上回来，哼爸买了粘老鼠的板子，不过看起来不咋好，10-1长假回来后家里出现老鼠了，真恐怖。

哼爸做的板栗炖肉真好吃啊，不过哼哼外婆捡的板栗也好，比较老比较香，表扬一下，呵呵

妇儿医院，你就一个忽悠

今天9月20日，提前预约好的羊水穿刺的日子

早上很早起来去做了验血和B超

下午排队去做羊水穿刺，穿好手术的衣服等进去了

我躺在手术台上，医生给我做了B超，找到羊水穿刺的地方，画上记号

医生也把长长的针插进去，羊水也抽了一些，然后令人气愤的事情发生了，而且那么恶心

医生告诉我：估计我之前保胎过，羊水抽出来也培养不了细胞，也就无法做检查的，没必要抽羊水做培养，在姐肚皮上还戳着一根长长的针

NND，预约的时候，你怎么不告诉姐如果保胎过，做穿刺也不能培养细胞呢

你说我请假去检查来做穿刺，我的妇幼保健册上都有中医院的单子，我也说过我前面有出血保胎过，而且里面有积血

你们在排羊水穿刺前也有拿着妇幼保健册上的单子看过，里面有我以前的B超单等，难道你都不看吗？

等人家担着风险千分之二的流产风险，上了手术台，都被你戳了一针，然后告诉不能继续，然后继续担着那1/250的先天愚型的风险，你这样不是很不负责的忽悠别人吗？

尽管给姐退了450元RMB，可是根本不是对姐负责，事后一护士还是医生说：别人保胎也有羊水抽出来是好的。可是里面的时候，那医生可不是这么说的

建议如果看到这篇博文的姐妹们，如果之前有保胎经历的姐妹，就算唐氏风险再高，也别被医生忽悠去做穿刺，免得和我一样杯具了
