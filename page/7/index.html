谷歌波波试用

上午在邮箱里发现了期待已久的谷歌波波邀请，是从一个网友那要来的。马上登录试用，第一印象是这东西怎么用啊？？看了一下《Google Wave从入门到进阶》，略微有点明白了：一个wave 就是一次能容纳文本、网页、视频、图片、Gadget（比如 Google doc） 等各类玩意儿的会话，而且是实时的，我觉得可以想像成一个很超前很超前的Gmail。Google 官方的说法已经表述的非常准确：Google Wave is an online tool for real-time communication and collaboration.

遗憾的是，现在Google wave 基本上不怎么好用。


  • 太慢，有时会感觉卡。

  • 占用资源太多，我用的 firefox内存占用到了从未出现过的 500MB。

  • 有些服务需要翻墙，比如 youtube 视频

此时我有个邪恶的想法，等 Google wave 正式发布以后，腾讯，百度啊赶紧上，整个山寨的出来，至少放在国内不用担心被墙。

GRUB4DOS 使用备忘

由于安装顺序的问题，比如，在已有 linux 的硬盘上安装 XP会覆盖 MBR，导致linux无法启动，又或者安装 linux 时没有把 grub 安装到 MBR，这时可以利用 grub4dos 来引导 linux。

GRUB4DOS 启动方式有很多种，参考《Grub4dos安装和启动》，这里是用windows 的启动管理器来启动。

从 GRUB4DOS 主页下载最新版，这里下载的是0.4.4版，解压并复制里面的 grldr 文件到C盘根目录，再修改 C:\boot.ini文件，增加一行:

c:\grldr="Start GRUB4DOS"
机子启动后，选择Start GRUB4DOS，自动会转到从盘的GRUB，这样双系统就可以正常使用。 如果想用linux 的 grub来引导，需要修改grub的menu.lst文件，增加windows引导选项，再用 grub-install 命令重新安装 grub 到MBR，命令类似于 
sudo grub-install /dev/sda

淘了一张电视卡


淘了一张电视卡，天敏的TB100，7130芯片，从黑龙江ems过来的邮费跟本身价格差不多，卡25￥，ems 23￥，淘宝上有总价更便宜的，不过我看中它附件比较齐全，包括了遥控器，所以还是比较满意的。本来想拍点照片，结果想起来的时候已经装到机箱里去了。-_-!!

上午去买了电视信号分配器、同轴线和一个F接头，一共12￥。

ubuntu 9.04 对7130支持得很好，将电视卡插上主板，连接好线缆，启动机子，然后打开 tvtime 软件搜索一下频道，就基本OK了。

有个比较恼人的问题：我把电视卡的audio-out 连到声卡的line-in，这样的话即使tvtime退出以后，电视声音还是存在，机子启动时还发出沙沙的噪音。目前的解决办法是把line-in静音绑定到F12键，退出tvtime时，按一下F12，启动tvtime时再按一下。不知道能不能用脚本实现。

推荐个电子书网站 &#8211; 皮皮书屋

在 TL 的邮件列表里潜水，有人说 皮皮书屋(www.ppurl.com)的用户注册机制很BT，还需要 C编译器。打开网站试着注册一下，会心一笑了。这种验证方式是给机器看的，而不是给人看的，哈哈。其实他的验证程序只有两份，多刷新几次，刷到简单的那份就行了，执行结果查查ascii表吧

浏览了一圈，里面大多是英文的计算机类书籍，包括最新的书籍，而且网站页面很简单，是典型的程序员架设的网站。

很好的网站，收藏了！
