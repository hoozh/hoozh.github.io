VS2005下利用 SVN 的 SubWCRev.exe 自增修订号

为了使软件版本号的最后一位，即修订号与由svn管理的代码库修订号保持一致并自动更新，可以利用VS2005的生成事件和 svn 中的 subwcrev 程序实现，步骤如下：


  1. 编写一个模板文件，比如取名为 version_template.h，内容如下
    #ifndef _VERSIONH
    #define _VERSIONH

    #define VER_MAJOR       1
    #define VER_MINOR       0
    #define VER_REVISION    $WCREV$

    #endif //!_VERSIONH


  2. 找到 subwcrev.exe 可执行文件，一般在 svn 客户端的安装目录下，比如 TortoiseSVN 是在其安装目录的bin文件夹下。这里，我将 subwcrev.exe 复制到了代码工程目录下，方便使用。

  3. 在VS2005的项目属性页->预生成事件->命令行中添加一句：
    $(SolutionDir)\subwcrev.exe $(SolutionDir) $(SolutionDir)\src\qmlv\version_template.h $(SolutionDir)\src\qmlv\version.h


    SubWCRev 调用语法:



    SubWCRev WorkingCopyPath [SrcVersionFile DstVersionFile]


  4. 这样，代码编译通过后，预生成事件被触发，VS2005 调用SubWCRev.exe程序，替换version_template.h中的宏$WCREV$为当前的svn修订号，并另存为 version.h文件。生成的 version.h 会类似于：
    #ifndef _VERSIONH
    #define _VERSIONH

    #define VER_MAJOR 1
    #define VER_MINOR 0
    #define VER_REVISION 2821

    #endif //!_VERSIONH


  5. 在程序中需要显示版本号的地方，引用 VER_REVISION 宏即可。

当然也可以用类似的办法，替换资源文件中的 ProductVersion 和FileVersion宏，这样就可以用系统函数 GetFileVersionInfo 函数来获取版本号。

俺的世博一日游

上周日去了一趟世博会，这么多年了，难得公司有一次福利，内牛满面…

记录一下吧，其实这种旅游一点也不舒服：

早上5点起床，5点半等车，9点多到上海世博园，从23号停车场步行15分钟至8号门，等候导游买票，经过九曲十八弯的铁栅栏等待安检，30分钟以上。

入园之后，不知道看哪个馆，瞎逛，这是因为我没做功课的关系 :) 由于我们一起的几个同事都不喜欢排队，所以我们只看等候时间不需太长的场馆，比如爱尔兰、突尼斯、希腊等等，亚洲馆基本没去，太远了。

后来又坐着渡轮晃到浦西，看了日本产业馆的八大企业的广告片，广告片本身不错，光影效果很好，幕布也不错，竖型幕布效果很震撼，瓷砖拼成的环形幕布很好玩，星空幕布很漂亮，可是我排了一个小时的队还是希望能看到一些实物的，嗯，我看到了一棵水培的松树和一只白色的乌龟。

再坐地铁回到浦东，基本上也就到了回去的时间了，顺路又看了一些小馆。在回去的大巴上，我觉得挺后悔的，我也许应该像同事甲一样排队4个小时看日本馆，或者像同事乙，排队3个小时看德国馆，这两个馆有我感兴趣的科技产品。

别样的建筑风格、异域风情确实不应该错过，也许以后很少会有这种机会去领略，但是人实在是太他母亲的多了，如果人流量降到10万人次/天，我一定再去玩一次。

最后上个销魂的背影吧

T410 使用记录

第一次用上小黑，做些记录

2010/6/12

天气越来越热，掌托部分感觉到热量，一方面不舒服，另外一方面心疼小黑，查了资料，据说是因为 NVS 3100 显卡 满负荷运行的关系，发热量太大。使用 nvidia system tools 可以调低显卡的工作频率，从而减少发热量，看链接。


  • 电源管理
    2010.4.14 更新的Power Manager 1.8 支持电池阈值设置，如果长期使用交流电，电池使用寿命会有所延长。
    当电池过分放电后，电池标尺可能不准确，可以通过“电池维护”中的“执行重新设置…”来恢复。
    待机一晚上，耗掉了15%的电力，以后还是休眠吧

  • 多点触控
    T410 的触控板是可以多点触控的，在看图片和网页的时候，放大缩小很方便，不过我个人还是习惯 ctrl++, ctrl+-一级一级缩放。另外触控板右边缘和下边缘可以模拟小红点+中键，水平和垂直滚动页面。

  • 指纹开机
    这功能太帅了，指纹冷开机，无需输密码直接进系统，而且识别率非常高，基本不会误判 。

  • 发热量
    个人觉得i5的CPU本身发热量还是挺厉害的，当然由于thinkpad优秀的散热设计，键盘上基本感觉不到热量。出风口比较热，特别是玩游戏或者外接显示器时。

  • USB 充电口
    右侧有一个黄色的USB口用于手机等设备充电，但是在关机的时时候只有插着交流电源，才有电压，要么就开机或待机才有电压，这个功能我感觉挺鸡肋的

  • 显卡问题
    之前不知是因为外接显示器切换频繁，还是双系统的关系，经常会出现系统正常启动，但是黑屏无显示，或显卡驱动不正常工作的情况。刷新     BIOS到最新版本后，没再出现以上问题。

Balsamiq Mockups &#8211; 一款手绘风格的原型设计软件


写界面软件的时候需要事先规划一下，但是 PhotoShop 太复杂，上次用过一次后就选择了用铅笔画个草稿，然后扫描一下存档，虽说非常方便，但是过段时间要想修改一下就麻烦了。

所以一直想找这一类的设计软件。搜索后发现这种软件叫做产品原型设计工具，而 Balsamiq Mockups 正是我所想要的，简单、清爽，控件丰富。引用一下韩军星的《Balsamiq Mockups完全手册》中提到的优点和特色。

可存：告别那一堆堆的纸质线框图吧…
易用：UI控件支持自动拖拽，并且可以实现自动对齐；
美观：手绘风格，简约清爽；
丰富：从按钮到输入框，从导航条到页面、表格，甚至包括了最潮的Iphone元素；
方便：对元素的修改除了工具栏外，还可以使用隐藏编辑框，支持快捷键；
兼容：可使用XML语言保存元素，也可以导出PNG图片，可以插入到任何项目；
跨平台：Windows、Mac OS、Linux下都可以使用，还有能集成在Confluence，JIRA，和XWiki中的版本，对了，如果你没时间下载桌面版，还可以直接使用网页版的Balsamiq Mockups…


试着用它绘制了一下我目前计划在做的互动系统学生端界面，还不错吧，我特别喜欢内置手写的英文字体，不过如果你希望输入中文字体，就需要勾选View 菜单下的 “Use System Fonts”，字体就没这么漂亮了。

另外还有在线版本，打开浏览器就能用


接下来要泼盆冷水了，这个软件要价79$，对个人来说相当贵，不过幸好还提供了一个功能有限的免费版本，限制如下：


  • 不能保存工程，但是可以导出/导入XML格式的数据，略微麻烦一些。

  • 桌面版不能导出PNG和PDF，但是在线版可以。

  • 每五分钟弹出一次对话窗口，提示这是免费版本。


比起 Axure PR 这种专业的软件，Balsamiq Mockups 功能没那么强大，但是我要求不高，画画示意图足够了。
