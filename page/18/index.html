引导问题

在台式机上顺利安装了slackware，想用grub来着，就用CD引导手动安装了一下，结果windows进不去了，又在笔记本上试，结果笔记本上windows也进不去了，真惨



刚才搜索了一下，发现是我的命令错误，我执行了: grub-install /dev/hda1



这样把grub安装在了/dev/hda1，覆盖了 windows 的引导信息，原本是应该安装在/dev/hda的，即



grub-install /dev/hda



怎么恢复呢，当然是重写或修复windows的引导信息喽



进入linux，root执行 :



dd if=/dev/hda1 of=/root/backupboot.hda1 count=1 skip=6

dd if=/root/backupboot.hda1 of=/dev/hda1 count=1



原理据说是只是覆盖了前面512字节的数据,拿后面的备份还原就可以了

试过，果然可以！





另外我在网上找到的方法还有：



用windows的安装盘引导，

进入故障控制台,输入一下命令:

c:>fixboot c:

c:>bootcfg /scan



不过没成功，显示 "disk error"

Slackware 下鼠标滚轮的问题解决

每次安装完Slackware都要设置一下鼠标滚轮，都快习惯了，但还是记不住语句怎么写，因为我根本不知道为什么-_-!



解决办法：

修改/etc/X11/xorg.conf文件



修改

Option "Protocol" "PS/2"

为

Option "Protocol" "IMPS/2"



增加一行

Option "ZAxisMapping" "4 5"

2007-04-01

最小化 OutLook 2003 到系统栏

开始->运行 regedit

展开 HKEY_CURRENT_USER/Software/Microsoft/Office/11.0/Outlook/Preferences”，

然后
右击空白区域，新建一个DWORD值“MinToTray”，并设键值为1，做完这些步骤，就可以最小化Outlook2003到系统托盘中了。

2007-03-16

猪啊猪


    看到 ysy 在sunu blog上的留言，窝才注意到今年春晚上没有提到“猪”字，本着学习的态度，google了一把，找到一篇文章http://www.nx.xinhuanet.com/tsnx/wzw/6.htm，摘录了一些。 



    很久很久以前，在阿拉伯半岛，因为气候炎热，且土地多为沙漠，不利于猪的生长和发展，猪的传染病很多，所以阿拉伯半岛就已经有禁食猪肉的习俗，古埃及人也视猪是不洁和厌恶之物。



    犹太教创立后，为了适应当时阿拉伯地区禁养猪的习俗，把禁食猪肉当作戒律写进圣经《旧约全书》中的《未利篇》第十一章中：“凡蹄分两半，倒嚼走兽，你们都可以吃……猪因为蹄分两半，却不反刍，就与你们不洁净。这些兽的肉你们不可吃，死的你们不可摸，都与你们不洁净。”



    唐宋时期，阿拉伯和波斯的穆斯林到中国经商，因他们信仰伊斯兰教，所以把禁食猪肉的习俗也带到了中国。经过几百年历史的发展，他们中的大多数人成为后来的回族。



    后来，回族在政治上受到压迫，回族的宗教信仰和风俗习惯也开始受到限制和歧视，甚至诬蔑。这说明明末回民不食猪肉的风俗习惯，已遭到统治阶级的破坏。



　　清代，回民不吃猪肉的风俗习惯更是受到嘲弄和污蔑。统治阶级在残酷镇压西北回民起义后，为了区分出回民与汉民，以是否吃猪肉来检验，以致引起更多回民的反抗与斗争。



    国民党统治时期，回民不吃猪肉同样遭到污辱。《南华文艺》公然刊载文章，造谣说回族清真寺里供有猪像，这是回族的祖先等等，引起了回民的强烈反抗，以至发展到全国回族的反抗运动。



    此后时期作者没提，省略…



    到了信息泛滥的时代，回民以猪为崇拜图腾的谣言依然通过网络，手机短信上传播。



    而此时，窝天朝正提出和谐社会的伟大概念，所以一切不和谐的字眼都是不应该出现的，以至于07年春晚没出现一个“猪”字。



    于是窝想，要是当年阿拉伯半岛的人们对猪不反感，也许今年赵本山大叔的小品就是“公猪下崽”了……

2007-03-04

想换手机了

前段时间，有很多人向窝抱怨通话质量不好，特别在甬江村，如果在房间里接电话，对方根本听不到声音，于是冬天的时候，窝总是很担心晚上有人打来电话，得跑到院子里接，一边抖一边说话。

琢磨着要换个手机。

现在的 Nokia 2100 用了两年半多，通话时间有101小时多一点，现在出现的问题除了通话质量不如以前，感觉待机时间也变短了。其他倒是没什么问题。不过窝依然相信Nokia的品质，毕竟它从床上掉下来好几次，要知道学校公寓的床比人还要高，低处的就更数不清次数了，而且窝也经常边充电边打电话。

还是想换个实用一点的，功能不必太多，但却完美的。举个反例，一个同学新买的国产手机，播放mp3，但声音调到最大时，有破音；短信彩信，GPRS功能齐全的狠，但是收到某些短信居然死机…

对窝来说，Nokia 2100 的优点在于：

  1. 通话质量好，当然不是指现在

  2. 屏幕清晰，反应速度快。哦，也许这是所有非彩屏手机优点

  3. 软件稳定，设置合理，简单

  4. 按键间隔大，舒服

  5. 皮实

  6. 大小合适

缺点：

  1. 存储空间小，短信、通讯录（且一个名字只有一个号码），备忘录

  2. 待机时间短，最高只有150小时。查了查 Philip 的9@9a, 1000小时，惊叹！只可惜功能太少

  3. 输入法，没有联想输入

  4. 闹钟只能设一个

  5. 卸载电池后要重设时间

都是细节的东西，先记录下来，比便日后参考。最近要搬家，也许信号变好，就不用急着换了
