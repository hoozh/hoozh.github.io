QQ 入驻 Win CE

新近发布的Windows Embedded CE 6.0 R3 包含以下新特性


  • Silverlight for Windows Embedded

    The power of Silverlight brought to Windows Embedded CE to create rich applications and user interfaces

  • Internet Explorer Embedded

    Internet Explorer with panning and zooming capabilities and a customizable interface to optimize the browsing experience on devices

  • Adobe Flash Lite

    Browser plug-in to render rich media Web sites

  • Touch and Gesture

    Plug-in engine to enable natural input capabilities and gesture animations

  • Connection Manager

    Infrastructure technology to manage multiple network interfaces on the device

  • Microsoft Office and PDF Viewers

    Applications to render Microsoft Office Word, PowerPoint, Excel and Adobe PDF content on the device

  • QQ Messenger

    Device-side client to connect to popular Instant Messaging service

也就是说，Win CE 6.0 上有腾讯官方授权的QQ软件了，根据cnBeta上的文章，界面挺丑，但是功能全面，支持视频聊天。

也就是说，国产神机M8上也能聊QQ了。

目前用着诺记的低端手机，通信是够了。不过还是希望能有个功能强大的手持设备，同步日历、任务、上网啥的。对 Wince 内核的手机系统不是很感兴趣，跟 iphone，android 系统相比，Windows Moblie 界面真是太土，虽然魅族自己开发的人机界面很不错，但似乎还是问题多多。

IJG 发布了 libjpeg v7

偶然打开 www.ijg.org， 发现 libjpeg 更新到了 Version 7 27-Jun-2009 ，当时我就震惊了。要知道上一个版本是 Version 6b 27-Mar-1998，这都快11年了。

libjpeg 是一个开源的 jpeg 图片编解码库，使用非常非常广泛。而且许多有名的图像库比如cximage, freeimage 都使用它来编解码 jpeg 图片。之前一个项目里用过6b版本的libjpeg，效率颇高，所以比较关注它。

研究一下源码包里的说明文件外加Google，发现 libjpeg 应该是由 Guido Vollbeding 接手维护了，主页是 http://jpegclub.org/。

ubuntu 9.04 上使用 dropbox

2009.10.21 Update: 发现也支持 apt-get 安装，具体过程。

继在windows上安装了dropbox后, 刚刚又在ubuntu上试了下，过程如下：


  1. 到主页上下载针对ubuntu9.04及相应平台的安装包，dpkg安装，或者像我一样双击安装，嘿

  2. 按照提示注销X

  3. 发现没有效果，在 console 执行 “dropbox status”, 结果：Dropbox isn’t running!

  4. 再执行“dropbox start”, 结果：
    Starting Dropbox…
    The Dropbox daemon is not installed!
    Run with -i to install the daemon

  5. 最后执行 “dropbox start -i”, 貌似自动下载了daemon并安装，dropbox可用了。


感谢 Himuraken

要说明的是：dropbox 是和 Nautilus 这个文件浏览器结合用的，所以在Nautilus里可以看到文件和文件夹上的状态图标(Overlay)，而KDE下的Dolphin里没有。虽然在Dolphin下也同样可以操作同步文件。

汽车转向灯开关的设计

上周五下午才开始学车，练了三天。练十天就去考科目二，教练还真是变态…

今天发现汽车转向灯设计的挺巧妙，原先我以为打了转向灯之后需要再手动拨回来，或者是过一段时间会自动拨回来。今天练车之后，发现是这样的：打了转向灯后，汽车转向，再把方向打直时，转向灯开关才会自动拨回来。可能是我见识少吧，觉着这样的设计确实方便很多。

试用 DropBox 小记


在这个时代用U盘同步数据确实够圡的，为了不至于太怀旧，开始试用Dropbox服务（其实主要原因是huaxlee的博客）。

DropBox 应该说是一个非常方便的网盘，安装完毕后 默认会在“我的文档”下生成一个名为“My DropBox”的目录，这个就是同步目录，当然也可以指定到别的地方。文件直接往里面扔就可以了。不同电脑安装DropBox后，文件自动更新同步。住处最近断网了，不然可以试试 linux 下效果如何。

同步速度挺快的，一般上传速度都有50-60KBps，快的时候200KBps，保存小文件基本上几秒钟就OK。单个文件通过PC端上传时不限制大小，通过 Dropbox 的网页上传要限制在350M以下。

初始空间2GB，邀请一个朋友，额外增加 256M 空间，被邀请人的空间同时也会增加，直到空间增加到 5 GB，所以感兴趣的朋友不妨通过我的邀请链接注册。

2009/08/17 流水帐

为了腾出硬盘空间装 Wince 6，打算刻录掉一些资料。正好电脑上没装刻录软件，于是开始找免费的刻录软件。Google 第一条就是 7款替代Nero的免费CD/DVD刻录软件下载（发现这种推荐性的文章搜索结果总是很靠前，比如 “N 款wordpress主题” ）。不知怎么搞的，选择了最后一个 Final Burner，结果证明我的选择是非常愚蠢的，这个软件在刻录的时候，不但界面会假死，而且刻录进度最后停在了224%，虽然我非常愿意相信它还在工作，不过等待的时候实在是太长了。强行结束程序，弹出光盘后仔细一看，大概刻了一半左右。

接着试用 Imgburn ，除了中文字体小了一号之外，很不错。刻了一张DVD，一张CD。

因为网络问题，wince6 总是有个包安装不上，明天再试试，或者直接找离线安装文件。
