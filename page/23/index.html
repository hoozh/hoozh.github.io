什么是luser？


最近新知道了一个词 － luser




 



wikipemdia 上的解释：



In Internet slang, a luser (sometimes expanded to
local user; also luzer or luzzer) is a painfully annoying, stupid, or
irritating computer user. It is a portmanteau of "loser" and "user" and
is usually pronounced as "loser". The word luser is often synonymous
with lamer. In hackish, the word luser takes on a broader meaning,
referring to any normal user (in other words not a guru), especially
one who is also a loser. This term can also signify the user as a
layman, as opposed to power user or administrator; for example, end
luser.



The Jargon File states that word was coined around 1975
at MIT. Under ITS, when you first walked up to a terminal at MIT and
typed control-Z to get the computer's attention, it printed out some
status information, including how many people were already using the
computer; it might print "14 users", for example. Someone thought it
would be a great joke to patch the system to print "14 losers" instead.
There ensued a great controversy, as some of the users didn't
particularly want to be called losers to their faces every time they
used the computer. For a while several hackers struggled covertly, each
changing the message behind the back of the others; any time you logged
into the computer it was even money whether it would say "users" or
"losers". Finally, someone tried the compromise "lusers", and it stuck.
Later one of the ITS machines supported "luser" as a request-for-help
command. ITS ceased to be used mid-1990, except as a museum piece; the
usage lives on, however, and the term "luser" is often seen in program
comments and on Usenet. /lusers is also a common IRC command to get the
number of users connected to a server or network. [1]



A
Luser Attitude Readjustment Tool (abbreviated LART) is "something
large, heavy and painful, used to respond appropriately to particularly
annoying lusers." A LART can by definition be anything from a pat on
the back, to a small tactical nuclear attack, depending on how much
adjustment the attitude needs. Other names for a LART include cluebat,
clue stick, Clueisville Slugger, clue-by-four and Board of Education.
See also Banhammer.




 




livid的“速成手册” ：



如果你想快速成为一个 luser，这里有一些提示可以帮助你上路。


1. 给电脑上安装了一个 Ubuntu，然后切换回 Windows，在 Baidu 中输入“Ubuntu 美化”。 2. 从来没有用过 gcc 也不想用。

3. 从来没有在 Unix 下运行过 man man。

4. 在论坛上发帖子问问题，把英文单词拼错，或者是使用同音汉字，甚至用比如 RPWT 之类的汉语拼音简写。

5. 在论坛的帖子的标题里头使用“跪求”之类的字眼。

6. 2001 年或者更早的时候就在电脑上安装过 Linux，但是至今还没有弄清楚究竟怎么装中文输入法。

7. 看到新的开源软件出来，还没有用过就想去帮人家做翻译。

8. 听到别人谈论 Java 然后就下意识地说“那个东西很慢”。

9. 听到别人在用 Windows 就下意识地说“那个东西很不安全”或者“那个是最好用的”。

10. 听到别人在用 Mac 就说“那是有钱人用的”或者“那个东西华而不实”。

11. 会用 C++ 写 Hello World，仅此而已，然后告诉全世界的人“我会 C++”。

12. 对成为 Python 或 PHP 高手抱有极大热情，然后上网找 CHM 格式的电子书却“懒得”去买一本 O'Reilly 的教材。

13. 硬盘里收集超过 1G 甚至更多的 CHM，当然了，不可能看完的。

14. 给别人发电子邮件时候使用 .doc 格式的附件。

15. 不知道甚至也从来没有听说过 luser 一词。



这个世界上实在是有太多太多的方式可以让你快速地成为一个 luser，所以，别灰心，成为 luser 是很容易的。

独自去绍兴

    大三的时候，曾经与人相约去绍兴，最后被放了鸽子，当时好不郁闷~。于是今年决定不找任何人，乘着国庆长假一个人去了绍兴。

　　

　　10月2号12点30从宁波火车站出发，1小时40分钟就到达了绍兴火车站，原来准备买第二天回宁波的车票的，后来看看长龙似的队伍便不管了，明天再说。

　　

　　出了火车站第一件事就是买一份绍兴地图，单身的自由就在于想去哪就去哪。第一站准备去鲁迅故里，找到7路公交车（两个方向的环线车），几站就到鲁迅路口，于是下车。

　　

　　无疑，鲁迅路位于绍兴城区的中心，是绍兴最繁华的地方，但比起宁波还是相差许多。在鲁迅路上闲逛，绍兴的建筑大部分有相同的风格，黑瓦，白墙，瓦与墙之间镶嵌一些暗红，这些在绍兴的老街上能看到更多，但是难免现代的因素总是搀杂其中，街上的店面如同绍兴的建筑一样，装修的古色古香，但是却不时听到用于吸引顾客震天的音乐。

　　

　　后来终于找到了位于鲁迅中路的鲁迅故里，先看到的是咸亨酒店，再进去就是鲁迅故居等等，考虑到时间问题，决定第二天再来，于是退回咸亨酒店，要了一碟茴香豆，一碟臭豆腐，半碗老酒。除了茴香豆，其他味道都还不错，老酒不敢多喝，怕晕。

　　

　　


    从咸亨酒店出来的时候，迎上一位三轮车夫，也就是传说中的本地导游，让他带着逛了逛绍兴的老街，主要在仓直桥街附近。在某些地方感觉还不错，特别是站在桥上看着街河，两岸都是黑瓦白墙，有女人在河边浣水洗衣，远处飘来几艘乌篷船，我想很多人都是冲着这个意境来绍兴的吧，后来又看了什么《冬至》的拍摄地冯家台门，黄酒馆，社戏戏台，没什么意思，于是回到事先定好的客栈住下。

　　

　　看了一会电视，觉得有点饿，决定出去找吃的。乱逛一周，发现青藤书屋就在附近，又去城市广场走了一圈，正好有个书市，买了两本书。从白天经过的仓直桥街回来，吃了两串臭豆腐，真是美味啊。

　　

　　第二天清早，退房。来到青藤书屋，那里正好刚刚开门，青藤书屋很小，出来一个前院，只有两间房，房后有个园庭，池子里放养几条着红鲤鱼。虽然很小，但对于文人来说应该足够了吧。

　　

　　接着走到鲁迅故里，门票很贵，加上人太多（最烦的就是人多），于是放弃了此程的目的，只在鲁迅故里的步行街上走了走，街上的所谓古玩在其他地方到处可见，太假太假。

　　

　　因为国庆，有两班旅游专线往返各个景点，先去兰亭，亭台楼阁、书法复制品、古碑，我不懂书法，所以不怎么感兴趣。

　　

　　接着到大禹陵，已经厌倦了随着人流到处走，于是决定步行去火车站，然后回宁波。查看地图，发现大禹陵在绍兴城区南面，火车站在北面，看起来很远，不过我喜欢这样。后来事实证明绍兴很小，不费多少力气就到了火车站。路上又去了戒珠寺、蔡元培故居。

　　

　　以上是流水帐，勉强看看吧。

　　

　　总结如下：

　　1.不太喜欢一个景点一个景点的去看，也许是不想做个普通的游客吧，以后要多学习一些旅游的经验，背个包，拿着地图，做个驴友走遍天下的感觉一定很不错。

　　

　　2.摄影技术实在是太烂啦～～

七月三十夜插地香


    刚才房东女儿和她表弟在往地上插香，问了问，原来是宁波的风俗，纪念地藏王菩萨。不过现在都是水泥地，只好插在下水道和阴井盖的地缝里咯。



    附由来：



    专管人间万物的地藏王，见人刀耕火种，生活艰难，看在眼里，急在心头，便对牛魔王讲：“侬介大力气在这里吃吃困困没意思，下凡帮人耕田去吧。”牛魔王摇头讲：“耕田我倒勿怕，怕就怕老了没力气给人一刀杀了当菜。”地藏王讲：“勿会，我敢担保。”牛魔王推三阻四，横竖不答应。地藏王急红了脸讲：“耕田原是侬的本职，咋好怕死不去。若是人真的杀了你，算我没长眼睛，你就来挖掉我的眼珠。”说着，双手一推，牛魔王勿防，从天上跌下来一头栽倒地上，上爿牙齿都敲光了，只好老老实实替人耕田。



    过了十多年，老牛耕不动田了，果然被人杀掉。牛魔王满肚皮怨恨，赶到天上找地藏王算帐，一把挖出地藏王眼珠，顺手抛出南天门。地藏王的眼珠滴溜溜落到江南水田里，变成田螺。玉皇大帝见地藏王瞎了眼，勿管能事，对牛魔王大发雷霆，把牛魔王痛骂一顿，罚其下凡做牛耕田，永远不准再回天廷。



    地藏王仍旧很关心人间疾苦，看不见就用耳朵贴在地上听人们说啥话、做啥事。观世音见他可怜，发了慈悲，每年七月三十送地藏王甘露一瓶洗眼，让他开眼一天。人们为了报答地藏王，每到这天夜里，家家户户都要插地香纪念他。

2006-08-22

养狗记


    男篮中国vs意大利那晚在快餐店看球，老板娘从外面揪着一只小狗进来，说拣的，估计是对面某大学生家的母狗生的，一胎九只，养不过来了。接过来玩一会儿，就在我手上睡着了，琢磨着：挺有缘的，养吧。就给抱回家去。



    只养过猫，还没养过狗，真不知道小狗是这么睡觉的：眼珠子常常翻白，还时不时的抽搐一下，吓坏了在场的一群人都以为快夭折。谁知睡了半小时，伸个懒腰，起来了，活蹦乱跳的。



    当晚，同居室友，即某男Z，坚决不睡地板。俺只有舍身取义，结果不但一宿没睡好，还被它从脚趾啃到了鼻子，与狗共枕，不亦困乎…



    周日，好酒好菜伺候，还得跟在它屁股后面收拾“残局”。不过经过我几番教导，有一次居然跑到厕所方便，房东女儿直夸乖巧。



    又夜，Z已离去，俺睡床上。半夜仍被吵醒，原来又要大便，早上发现虽然还是拉在院子中央，俺已经很感激了…



    周一上班，琢磨着关房间不好，就放院子里。到办公室后，听一后来的同事（住在我附近）说看到一小狗跟着一老太太和小孩走，心想：没了。



   于是，从此永别了，sigh

我们就是学这个的


    与某男Z，拿了两个电风扇去修理



    修理师傅拿出一个万用表测了测其中一个电扇的插头，看看数字，说到：“这个估计电机烧坏了”。拆开再测，还是同一结论，于是放弃。修另外一个，这个还好，只是电容击穿了，于是换了一个电容，晕了一个多月的电扇又活过来了。于是交钱，10块（我还跟师傅要了一个电容）。



    此时，某男Z说出一句话来：“laoz,我们拜师学艺吧”。注意：此男“历经坎坷”，最近刚签下一份工作。 我想了想，说：“我们就是学这个（电子信息科学与技术）的”。



    出门之后，那个师傅还在笑：“嘿嘿，他们就是学这个的”。
