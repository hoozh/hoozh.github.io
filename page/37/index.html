[LFS]时区设置

参见hints中的time.txt

1.决定使用UTC还是local time
  因为要与windows共存,我选择local time
  修改/etc/sysconfig/clock文件
  UTC=0 #表示local time,=1则是UTC

2.判断时区 
  运行 tzselect,回答2到3个问题,判断时区
  北京时间当然是 Asia/Shanghai了,记下

3.修改/etc/localtime文件
  cp –remove-destination /usr/share/zoneinfo/Asia/Shanghai /etc/localtime

4.检查boot-scripts，/etc/rc.d/init.d/setclock
  LFS6.0里的和time,txt写法略微不同，功能一样，不作修改
  注意setclock权限为 755

5.建立 symlinks
# ln -s ../init.d/setclock /etc/rc.d/rc0.d/K45setclock &&
# ln -s ../init.d/setclock /etc/rc.d/rc6.d/K45setcloc

6.TZ环境变量
一些程序会依赖于时区的设置,hints里写的 :)
在/etc/profile中加入一行

export TZ=Asia/Shanghai

Install zhcon-0.2.3 from source

I used to install zhcon with .tgz on Slackware Linux or rpm on other Linux, because I never sucessed from source :(

now I find the patch

download: 

zhcon-0.2.3:
http://prdownloads.sourceforge.net/zhcon/zhcon-0.2.3.tar.gz?download

zhcon-0.2.3-gcc34.patch.txt:  
http://www.linuxsir.org/bbs/attachment.php?attachmentid=20116

Install:
if you use gcc 3.4.x you will have some problems like:
fblinear24.cpp: In member function `virtual void FBLinear24: rawChar(int, int, int, int, CharBitMap)’:
fblinear24.cpp:206: error: ISO C++ forbids cast to non-reference type used as lvalue
fblinear24.cpp:210: error: ISO C++ forbids cast to non-reference type used as lvalue
make[3]: ** [fblinear24.o] 错误 1
make[2]:  [all-recursive] 错误 1
make[1]:  [all-recursive] 错误 1
make: * [all] 错误 2 

so you need the patch above

install zhcon using below commands

$ mv zhcon-0.2.3-gcc34.patch.txt zhcon-0.2.3-gcc34.patch 
$ tar zxvf zhcon-0.2.3.tar.gz
$ cd zhcon-0.2.3
$ patch -Np1 -i ../zhcon-0.2.3-gcc34.patch.txt &&
  ./configure –prefix=/usr/local &&
  make 
# make install


————————————————
tgz for Slackware Linux

http://download.linuxsir.org/nbxmedia/slackware-updates/ap/zhcon-0.2.3-1.i386.tgz

slax 定制的脚本runme.sh

#!/bin/bash
#
# run this script to create a LiveCD in /tmp/livecd.iso
# Your kernel image has to be in $ROOT/boot/vmlinuz or $ROOT/vmlinuz
#

export PATH=.:./tools:../tools:/usr/sbin:/usr/bin:/sbin:/bin:/

# 确定在linux-live目录
CHANGEDIR=”dirnamereadlink -f $0`"<br />echo "Changing current directory to $CHANGEDIR"<br />cd $CHANGEDIR<br /><br /># <br />. liblinuxlive || exit 1<br />. config || exit 1<br /><br /># 复制所需程序到/usr/sbin /usr/lib 见 ./install 文件<br />./install $ROOT<br /><br /># 内核<br />VMLINUZ=$ROOT/boot/vmlinuz<br />if [ -L "$VMLINUZ" ]; then VMLINUZ=readlink -f $VMLINUZ; fi<br />if [ "ls $VMLINUZ 2>/dev/null`” = “” ]; then echo “cannot find $VMLINUZ”; exit 1; fi

header “Creating LiveCD from your Linux”

echo “Using kernel from $VMLINUZ”
echo “Using kernel modules from /lib/modules/$KERNEL”

# 制作initrd 镜像
echo “creating initrd image…”
cd initrd
./initrd_create
if [ “$?” -ne 0 ]; then exit; fi
cd ..

# 建立光盘目录和复制文件
mkdir -p $CDDATA/base
mkdir -p $CDDATA/modules
mkdir -p $CDDATA/optional
mkdir -p $CDDATA/devel

echo “copying boot files to $CDDATA…”
cp initrd/$INITRDIMG.gz $CDDATA/initrd.gz
rm initrd/$INITRDIMG.gz
cp -R bootfiles/* $CDDATA
cp -R {info,tools} $CDDATA

# 压缩dir，/etc /var /usr 等
echo “creating compressed images…”

for dir in bin etc home lib opt root usr sbin var; do
if [ -d $ROOT/$dir ]; then
echo “base/$dir.mo”
create_module $ROOT/$dir $CDDATA/base/$dir.mo -keep-as-directory
if [ $? -ne 0 ]; then exit; fi
fi
done

echo “copying kernel from $VMLINUZ…”
cp $VMLINUZ $CDDATA/vmlinuz

制作iso
echo “creating LiveCD ISO image…”
cd $CDDATA
./create_bootiso.sh /tmp/livecd.iso

cd /tmp
header “Your ISO is created in /tmp/livecd.iso”
