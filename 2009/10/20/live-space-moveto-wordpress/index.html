live space 搬家到 wordpress

把 live space 上的几篇带鱼文搬到了这里，用的是 Live Space Mover，非常有效。文章和留言全部搬过来了，唯一的问题就是带入了多余的分类，手工修正一下即可。

具体使用方法见下面两篇文章：


  • http://b2.broom9.com/?page_id=519 （官方英文）

  • LiveSpace的搬家利器：Live Space Mover


开始的时候遇到了问题，程序提示：

LINE 107 : WARNING Can’t find date
LINE 564 : ERROR Unexpected error
Traceback (most recent call last):
File “live-space-mover.py”, line 562, in <module>
main()
File “live-space-mover.py”, line 485, in main
i=fetchEntry(permalink,datetimepattern,mode)
File “live-space-mover.py”, line 108, in fetchEntry
sys.exit(2)
SystemExit: 2

仔细查看后发现是 livespace 的设置没弄好，研究之后搞定了，以下是我的设置：

共享空间 ► 选项 ► 常规

共享空间 ► 选项 ► 权限

共享空间 ► 选项 ► 日志

对了，我用的 Live Space Mover 是 1.7.5 版。
