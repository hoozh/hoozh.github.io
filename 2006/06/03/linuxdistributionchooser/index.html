Linux Distribution Chooser

选择一个适合自己 Linux 发行版一向是个令人头疼的问题，挪威zegenie studios 的 Daniel Andre 准备了一个选择合适 Linux 发行版的小测验 ，有兴趣可以试试看。

这个测验包括了 17 个 Linux 发行版，有：Fedora Core, Mandriva, SuSE,
Debian, MEPIS, Ubuntu, Kubuntu, Linspire, Xandros, PCLinuxOS, Yellowdog
Linux, Gentoo, Slackware, Knoppix, OpenSuSE, ArchLinux, 和 Foresight
Linux.，通过向你询问几个问题，为你选出最合适你的版本。

因为我喜欢从源码编译软件包，所以它为我选择了 Gentoo 和 Slackware。
