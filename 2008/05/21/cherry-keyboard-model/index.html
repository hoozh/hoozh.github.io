Cherry 键盘型号说明

举例说明：

G80-3000#$%&&

（以符号代表每个字母的位置）

# = 印字技术

L = 雷射刻字 H = 二色成型

$ = 轴类

P = 黑轴

Q = 白轴

S = 青轴

K = 茶轴

% = win键

A = Windos Key 无

M = Windos Key 有

&& = 配列

US = English

EU = Euro

DE = German

BE = Belgian

CH = Swiss

DK = Dansih

ES = Spanis

FR = French

举例:

Cherry G80-3000 LQMEU L=雷射刻字, Q=白轴, M=Windows Key有, EU=Euro

Cherry G80-3000 LPMEU L=雷射刻字, P=黑轴, M=Windows Key有, EU=Euro

Cherry G80-3000 LSMEU L=雷射刻字, S=青轴, M=Windows Key有, EU=Euro

Cherry G80-3484 HKCUS （因为3484是特制的键盘型号，但从型号同样看出来规格）
